Exercise 1: Write a program that takes a number from the input, compares it to 10, and returns a message for each of the 3 modes (larger, equal, and smaller).

برنامه ای بنویسید که عددی را از ورودی گرفته، با 10 مقایسه کند و برای هر یک از 3 حالت (بزرگتر، مساوی و کوچکتر) پیامی برگرداند

#!/bin/bash

# درخواست عدد از کاربر
echo "لطفاً یک عدد وارد کنید:"
read number

# مقایسه عدد با 10
if [ "$number" -gt 10 ]; then
    echo "عدد وارد شده بزرگتر از 10 است."
elif [ "$number" -lt 10 ]; then
    echo "عدد وارد شده کوچکتر از 10 است."
else
    echo "عدد وارد شده مساوی با 10 است."
fi
chmod +x compare.sh
./compare.sh

Exercise 2: Write a program that takes 20 numbers from the input, compares them, and states which number is the largest and smallest.

برنامه ای بنویسید که 20 عدد را از ورودی گرفته و آنها را با هم مقایسه کند و بگوید کدام عدد بزرگترین و کوچکترین است

#!/bin/bash

# درخواست 20 عدد از کاربر
echo "لطفاً 20 عدد وارد کنید:"

# خواندن اعداد و ذخیره آنها در یک آرایه
read -a numbers

# بررسی اینکه تعداد اعداد وارد شده دقیقاً 20 باشد
if [ "${#numbers[@]}" -ne 20 ]; then
    echo "لطفاً دقیقاً 20 عدد وارد کنید."
    exit 1
fi

# فرض اینکه اولین عدد بزرگترین و کوچکترین است
max=${numbers[0]}
min=${numbers[0]}

# بررسی بقیه اعداد
for num in "${numbers[@]}"; do
    if [ "$num" -gt "$max" ]; then
        max=$num
    fi

    if [ "$num" -lt "$min" ]; then
        min=$num
    fi
done

# نمایش بزرگترین و کوچکترین عدد
echo "بزرگترین عدد: $max"
echo "کوچکترین عدد: $min"
chmod +x find_min_max.sh
./find_min_max.sh

Exercise 3: Write a program that has the IP of a server and its User/Pass in front of the Script name and if it is pingable, sends its /etc/passwd file to /home/user path of that server, otherwise a message displayed that the server is not accessible.

برنامه ای بنویسید که IP یک سرور و User/Pass آن را در جلوی نام اسکریپت داشته باشد و اگر قابل پینگ باشد، فایل /etc/passwd خود را به مسیر /home/user آن سرور ارسال می کند، در غیر این صورت پیامی نمایش داده می شود که سرور قابل دسترسی نیست

#!/bin/bash

# گرفتن IP، نام کاربری و رمز عبور از نام اسکریپت
if [ "$#" -ne 3 ]; then
    echo "استفاده: $0 <IP> <User> <Pass>"
    exit 1
fi

SERVER_IP=$1
USERNAME=$2
PASSWORD=$3

# پینگ کردن سرور
ping -c 1 $SERVER_IP > /dev/null 2>&1

if [ $? -eq 0 ]; then
    echo "سرور قابل دسترسی است. ارسال فایل /etc/passwd به /home/$USERNAME در سرور..."

    # ارسال فایل با استفاده از sshpass
    sshpass -p "$PASSWORD" scp /etc/passwd $USERNAME@$SERVER_IP:/home/$USERNAME/

    if [ $? -eq 0 ]; then
        echo "فایل با موفقیت ارسال شد."
    else
        echo "خطا در ارسال فایل."
    fi
else
    echo "سرور قابل دسترسی نیست."
fi


sudo apt-get install sshpass
chmod +x send_passwd.sh

./send_passwd.sh <IP> <User> <Pass>

Exercise 5:Write a program that saves the first and third fields of the /etc/passwd file every day in a file with the same date and does not hold it for more than two days.

برنامه ای بنویسید که فیلدهای اول و سوم فایل /etc/passwd را هر روز در فایلی با همان تاریخ ذخیره کند و بیش از دو روز آن را نگه ندارد.

#!/bin/bash

# تنظیم متغیر تاریخ
TODAY=$(date +%Y-%m-%d)
FILENAME="/path/to/save/${TODAY}_passwd.txt"

# استخراج فیلدهای اول و سوم و ذخیره در فایل
awk -F: '{print $1, $3}' /etc/passwd > "$FILENAME"

# حذف فایل‌های قدیمی‌تر از دو روز
find /path/to/save/ -type f -name '*_passwd.txt' -mtime +2 -exec rm {} \;
chmod +x backup_passwd.sh

برای اجرای اسکریپت به صورت روزانه، از cron استفاده کنید. برای این کار، دستور زیر را اجرا کنید تا ویرایشگر crontab باز شود:

crontab -e

خط زیر را به crontab اضافه کنید تا اسکریپت هر روز در ساعت 00:00 اجرا شود

0 0 * * * /path/to/backup_passwd.sh

Exercise 6: Write a program that take a backup from home directory of your user after each time user logged out.

برنامه ای بنویسید که پس از هر بار خروج کاربر از فهرست اصلی کاربر شما یک نسخه پشتیبان تهیه کند.

#!/bin/bash

# تنظیم مسیرهای مبدا و مقصد
SOURCE="/home/your_username/"
DEST="/path/to/backup/directory/"

# تهیه نسخه پشتیبان با استفاده از rsync
rsync -av --delete "$SOURCE" "$DEST"
chmod +x /path/to/backup_script.sh

سپس، در فایل .bash_logout کاربر خود (مثلاً /home/your_username/.bash_logout)، خط زیر را اضافه کنید تا اسکریپت پشتیبان‌گیری هر بار که کاربر از شل خارج می‌شود، اجرا شود:

# اجرای اسکریپت پشتیبان‌گیری
/path/to/backup_script.sh

Exercise 7: Write a program that reads, pings one by one from within a file containing the list of destination IPs, and saves the result in a log file on the same day with the hostname of that machine.

برنامه ای بنویسید که بخواند، یک به یک از داخل یک فایل حاوی لیست IP های مقصد پینگ کند و نتیجه را در یک فایل گزارش در همان روز با نام میزبان آن ماشین ذخیره کند

ابتدا فایل حاوی لیست IP ها را ایجاد کنید. مثلاً در مسیر /path/to/ip_list.txt:

8.8.8.8
8.8.4.4
1.1.1.1

#!/bin/bash

# تنظیم مسیر فایل ورودی و فایل گزارش
IP_LIST_FILE="/path/to/ip_list.txt"
LOG_DIR="/path/to/logs"
HOSTNAME=$(hostname)
TODAY=$(date +%Y-%m-%d)
LOG_FILE="${LOG_DIR}/${HOSTNAME}_${TODAY}_ping.log"

# ایجاد دایرکتوری گزارش در صورت عدم وجود
mkdir -p "$LOG_DIR"

# خواندن فایل IP و پینگ کردن هر IP
while IFS= read -r ip; do
    if ping -c 1 "$ip" &> /dev/null; then
        echo "[$(date)] $ip is reachable" >> "$LOG_FILE"
    else
        echo "[$(date)] $ip is not reachable" >> "$LOG_FILE"
    fi
done < "$IP_LIST_FILE"

مراحل اجرای اسکریپت:
فایل حاوی لیست IP ها (/path/to/ip_list.txt) را با IP های مقصد پر کنید.
اسکریپت Bash را در یک فایل (مثلاً ping_script.sh) ذخیره کنید و مسیرهای IP_LIST_FILE و LOG_DIR را به مسیرهای مناسب برای سیستم خود تغییر دهید.

chmod +x ping_script.sh

./ping_script.sh

توضیحات اسکریپت:
IP_LIST_FILE: مسیر فایل حاوی لیست IP ها.
LOG_DIR: مسیر دایرکتوری برای ذخیره فایل‌های گزارش.
HOSTNAME: نام میزبان ماشین فعلی.
TODAY: تاریخ فعلی به فرمت YYYY-MM-DD.
LOG_FILE: مسیر فایل گزارش با نام میزبان و تاریخ فعلی.
mkdir -p "$LOG_DIR": ایجاد دایرکتوری گزارش در صورت عدم وجود.
while IFS= read -r ip; do ... done < "$IP_LIST_FILE": خواندن فایل IP خط به خط.
ping -c 1 "$ip" &> /dev/null: پینگ کردن هر IP. اگر قابل دسترسی بود، پیام "reachable" و در غیر این صورت پیام "not reachable" را در فایل گزارش می‌نویسد.
